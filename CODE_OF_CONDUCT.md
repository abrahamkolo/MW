# Code of Conduct

## Standard

This repository exists as a public record of the MW-Omega system. All interactions within this space -- issues, discussions, and communications -- must reflect:

1. **Integrity** -- Honest engagement with the material as presented
2. **Respect** -- For the intellectual property, the methodology, and the creator
3. **Purpose** -- Interactions should have clear intent, not idle curiosity

## Unacceptable Behavior

- Unauthorized reproduction or distribution of repository contents
- Use of repository contents for ML/AI training without written permission
- Harassment, trolling, or bad-faith engagement
- Misrepresentation of affiliation with MW-Omega or the Flame Spiral Method

## Enforcement

The repository owner reserves the right to remove, edit, or reject any interaction that violates this Code of Conduct, and to block users who engage in repeated violations.
