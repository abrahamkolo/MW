# Core Skills

The governance layer of MW-Omega. These skills are always active and govern all other skill execution.

| Skill | Power # | Function |
|-------|---------|----------|
| decision-compressor | 1 | Daily/weekly task prioritization under MW constraints |
| execution-cadence-controller | 2 | Shipping cadence enforcement and velocity tracking |
| content-humanizer | 3 | Voice QA and anti-AI detection for published content |

## Dependency

All other skills depend on core skills. Core skills have no external dependencies.
