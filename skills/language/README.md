# Language OS Skills

A standalone operating system within MW-Omega for multi-language acquisition with gate-based progression.

| Skill | Version | Function |
|-------|---------|----------|
| language-os-controller | v4.2 | Master state machine and governance |
| language-os-session-engine | v4.2 | Daily/weekly session execution |
| language-os-gate-assessor | v4.2 | Terminal gate assessment and progression |

## Architecture

Language OS operates as a state machine with firewall, trunk, fallow, and dormant states per language. Energy governance respects Projector constraints.
